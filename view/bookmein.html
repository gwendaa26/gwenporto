Book Me In Application

An application to ease the users to book the appointment based on the existing services

Book Me In is a booking mobile application based on iOS platform. In here, the server provider offers their services to book the appointmment. After that, the end user will accept the confirmation of booking appointment after selecting the services. Then, the calendar will automatically update for both service provider and end user. The purpose of this application is to aid the end user to find the services that the service providers have already delivered and suit for them to book the appoitnment.

Regarding to the project, it can be said nearly complete. A few features cannot work appropriate due to the database problems that couldn't receive to the code at the moment. Therefore, this application is still in offline mode, which cannot show the data from the database or a server.

At last, I had two roles to contribute in this project. There were a mobile developer and a UI/UX Designer. As a mobile developer, I did the codes for the interface in this application. For instances :

- Login page
- Register page
- Homepage
- List of services
- Description of services
- Appointment confirmation page
